例如：不应当假设资源一定支持续传，要首先使用 request header range 检查目标资源是否支持续传，支持 range 的 http resonse code 是 206。

另外，两次断点下载期间，资源可能发生变化，需要在请求时同时附带上 Etag 或 Last-modified 记录，由服务器确定资源是否变化。
