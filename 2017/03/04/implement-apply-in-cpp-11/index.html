Why apply matters

  • 客户代码的请求结束时的回调 callback/handler
  • 针对具体业务的 response parser；因为 HTTP 是文本协议，无论 response data 基于何种格式（例如 JSON），我们都要针对具体的业务逻辑场景进行解析
